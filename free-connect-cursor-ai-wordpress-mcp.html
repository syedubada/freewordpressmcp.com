How to Connect Cursor AI to WordPress via MCP

Use Cursor AI to analyze your WordPress website, create blog posts, improve SEO rankings, clean up spam comments, and build stylish Elementor layouts directly from your AI agent without manual copy-pasting.

What You Can Do with Cursor AI + WordPress MCP

Cursor AI is a powerful AI-first code editor and agent. When connected to WordPress through the Model Context Protocol (MCP), Cursor can communicate directly with your site's REST API:

  • Draft and publish fully structured, SEO-friendly blog posts directly into WordPress
  • Audit existing blog posts and pages to identify search engine optimization gaps
  • Automatically generate Elementor pages and custom widget structures
  • Read, moderate, and bulk clean up spam comments in your WordPress dashboard
  • Manage pages, taxonomies, categories, tags, and media library assets
  • Eliminate manual copy-pasting between Cursor AI and your WordPress dashboard

Generated Elementor pages currently use free Elementor widgets and pull responsive placeholder images from Unsplash — a solid starting layout you can refine further in the Elementor editor. Support for Elementor Pro and other paid widgets is on the roadmap for the plugin.

How to Connect Cursor AI with WordPress: Step-by-Step

01

Download the WSP MCP Plugin

You now only need a single plugin. On the WSP WordPress MCP GitHub, open Releases and download wsp-mcp-ai-agents-connector.zip from the latest release (v2.3.0 at recording). The MCP Adapter is bundled inside, so there's no separate adapter plugin to install anymore. Always grab the newest release.

02

Install & Activate the Plugin in WordPress

In WP Admin, go to Plugins → Add New → Upload Plugin. Upload the ZIP and click Activate Plugin. A new "MCP" menu appears in your sidebar.

03

Configure Abilities & Elementor Permissions

Go to MCP → Settings. Toggle on the abilities you want Cursor AI to perform (Posts, Pages, Taxonomy, Comments, Media, Users, Search). Note: Elementor abilities only appear after the Elementor plugin is installed and active. Click Save Settings.

04

Copy Cursor JSON Config Snippet

Go to MCP → Config Files, select the Cursor tab, and click Copy to Clipboard.

05

Open MCP Settings in Cursor AI

Launch Cursor AI, click the gear icon in the bottom left to open Settings, navigate to Tools & MCP, and click Add Custom MCP (or edit mcp.json).

06

Paste Config & Save the File

Paste the copied JSON into mcp.json and save the file. The snippet already includes your site's connection credentials, so there's nothing to replace.

07

Verify Connection & Run Prompts

Verify that the WordPress MCP server shows as enabled under Cursor's MCP tools. Open a New Agent chat in Cursor and prompt: "Please create a new blog post related to how to rank my website in Google."

Frequently Asked Questions

Is connecting Cursor AI to WordPress free?

Yes. The WSP MCP – AI Agents Connector plugin (which bundles the MCP Adapter) is 100% free and open-source under the MIT license. Cursor AI offers a free plan with built-in MCP support.

Can Cursor AI create Elementor layouts directly inside WordPress?

Yes. When connected via MCP, Cursor AI can generate Elementor pages, add widgets, sections, and layout structures directly on your WordPress site.

Do I need coding skills to connect Cursor AI with WordPress?

No coding skills are required. You install one plugin, then copy a ready-made JSON config from WordPress into Cursor's mcp.json settings — no manual credentials to set up.

What can Cursor AI do on my WordPress website?

Cursor AI can draft and publish blog posts, audit existing content for SEO, clean up spam comments, manage pages, and build Elementor layouts through natural language prompts.
