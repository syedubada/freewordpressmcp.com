Free - Connect Claude AI With WordPress MCP

Automate your WordPress website using Claude AI — completely free. Create blog posts, manage pages, and handle WooCommerce products through natural language chat. No coding required.

What You Can Do with Claude AI + WordPress MCP

Once connected, Claude AI becomes your personal WordPress assistant — controlled entirely through natural language chat. No dashboard clicking, no hiring a developer, no copying and pasting between tools. Here's what you can do from inside the Claude interface:

  • Create SEO-friendly blog posts automatically, complete with H2/H3 headings and proper content structure
  • Create, update, and delete WordPress pages without ever opening the dashboard
  • Add and manage WooCommerce products using plain English descriptions
  • Audit existing posts for SEO issues and get actionable improvement suggestions
  • Analyze your entire WordPress content library for quality and keyword coverage
  • Schedule content creation tasks using Claude's built-in automation features
  • Check your site for potential security issues and malware signals
  • Move posts to trash or restore them — all from a single chat window

This is powered by the Model Context Protocol (MCP) — an open standard created by Anthropic that lets AI models connect directly to external tools and APIs. The free WSP WordPress MCP plugin exposes your WordPress capabilities to Claude in a structured, permission-controlled way so you stay in full control.

How to Connect Claude AI with WordPress: Step-by-Step

01

Download the MCP Adapter Plugin

The MCP Adapter is the bridge between your WordPress REST API and Claude's MCP protocol. Go to the MCP Adapter GitHub repository, click Releases, and download the latest .zip file.

02

Download the WSP WordPress MCP Plugin

This free open-source plugin by WebSensePro exposes granular WordPress abilities to Claude AI. Download the latest release from the WSP WordPress MCP GitHub repository. No account required — it's MIT licensed.

03

Install and Activate Both Plugins in WordPress

In your WordPress dashboard, go to Plugins → Add Plugin → Upload Plugin. Upload and activate the MCP Adapter first, then the WSP WordPress MCP plugin. After activation, you'll see a new "MCP" menu item appear in your sidebar.

04

Configure Which Abilities Claude AI Can Use

Click MCP → Settings. Toggle on the specific abilities you want Claude to have: create posts, read posts, delete posts, manage pages, WooCommerce products, comments, media, users, and more. Toggle off anything you don't want Claude to touch — you stay in complete control.

05

Copy the Claude AI Config File

Go to MCP → Config Files and click "Copy to Clipboard" under the Claude AI section. This JSON snippet tells Claude your WordPress URL and which ability endpoints to use — it's what connects the two systems together.

06

Generate a WordPress Application Password

Go to Users → All Users → Edit your admin user. Scroll to Application Passwords, create a new one (name it "Claude"), copy it, and replace the replace with your application password placeholder in the config file. Keep this password private — it grants full API-level access to your site.

07

Paste the Config into Claude and Restart

Open Claude → Settings → Developer → Edit Config. Paste the config JSON, save the file, then fully quit and reopen Claude. Return to Settings → Developer — you should see the WordPress MCP server listed and active. A red error means something in the config needs rechecking.

08

Test It — Ask Claude to Create a Blog Post

Open a new Claude chat and type: "Please create a new SEO-friendly blog post in my WordPress website about [your topic]." Claude will request permission on first use — click Always Allow. Within seconds you'll have a fully structured, heading-tagged, SEO-ready blog post live in your WordPress dashboard.

Frequently Asked Questions

Is connecting Claude AI to WordPress free?

The WSP WordPress MCP plugin and the MCP Adapter are both 100% free and open source. Claude AI's desktop app is free to download. Claude's AI subscription starts at $20/month for full usage — the free tier has usage limits.

Do I need coding skills to connect Claude AI with WordPress?

No coding knowledge is required. This is a beginner-friendly setup that involves installing two plugins, copying a config file, and generating an application password — it takes about 5–10 minutes from start to finish.

What can Claude AI do with my WordPress website?

Claude AI can create SEO-optimized blog posts, create and update pages, manage WooCommerce products, read and audit existing content, schedule tasks, delete posts, and check for security issues — all through a natural language chat interface.

Is it safe to give Claude AI access to my WordPress site?

Yes, with proper precautions. The WSP WordPress MCP plugin gives you granular ability controls — you can restrict Claude to read-only or enable only specific post types. Always keep your application password private and revoke it if you suspect it has been compromised.

What is MCP (Model Context Protocol)?

MCP (Model Context Protocol) is an open standard developed by Anthropic that allows AI models like Claude to securely connect with external tools, databases, and APIs. It's the protocol that lets Claude "talk" to your WordPress site without needing direct server access.

Does this work with WooCommerce?

Yes. The WSP WordPress MCP plugin includes WooCommerce support. You can ask Claude to create products, manage categories, and interact with your store content directly from the Claude chat interface.

Do I need to say "use WSP MCP" in every Claude chat message?

No — once the MCP server is connected, Claude automatically knows it has WordPress access. Simply say "create a blog post in my WordPress site" and Claude will use the MCP tools without needing a manual reminder.

What happens if Claude AI has red status in Developer settings?

A red status means the connection failed. The most common causes are: a typo in the application password, the wrong WordPress site URL, or one of the plugins not being activated. Re-check these three things and restart Claude.
