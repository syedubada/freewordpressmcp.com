How to Connect WordPress MCP to Google Antigravity 2.0

Connect Google Antigravity IDE & Agent directly to your WordPress site. Automate Elementor layouts, publish blog posts, audit SEO, and clean up spam — directly through AI chat without manual copy-pasting.

What You Can Do with Google Antigravity 2.0 + WordPress MCP

Google Antigravity 2.0 combines an IDE and an AI Agent framework. When linked to your WordPress site using Model Context Protocol (MCP), Antigravity can interact directly with your site's backend:

  • Automatically generate high-converting, stylish Elementor layouts and hero sections
  • Create, update, and publish blog posts and landing pages directly into WordPress
  • Audit site SEO, analyze content quality, and optimize meta titles and descriptions
  • Bulk manage comments, moderate pending approvals, and clean up spam
  • Manage pages, media libraries, user roles, and taxonomy tags with natural language prompts
  • Eliminate manual copy-pasting between Antigravity and your WordPress Dashboard

How to Connect Google Antigravity 2.0 with WordPress: Step-by-Step

01

Download Required Plugins

Download the latest release ZIPs for both mcp-adapter.zip (from WordPress MCP Adapter GitHub) and wsp-wordpress-mcp.zip (from WebSensePro GitHub repository).

02

Install & Activate Plugins in WordPress

In WP Admin, navigate to Plugins → Add New → Upload Plugin. Upload and activate both ZIP files. You will see a new "MCP" menu in your sidebar.

03

Enable Abilities & Elementor Permissions

Go to MCP → Settings. Turn on the abilities you want Antigravity to perform (Posts, Pages, Media, Comments, and Elementor abilities). Click Save Settings.

04

Copy Antigravity JSON Config Snippet

Go to MCP → Config Files, select the Antigravity tab, and click Copy to Clipboard.

05

Generate WordPress Application Password

Go to Users → All Users → Edit your Admin profile. Scroll to Application Passwords, enter a name (e.g. Antigravity), click Add New Application Password, and copy the generated password string.

06

Open MCP Configuration in Antigravity IDE

Launch Antigravity IDE, click the sidebar toggle to open the Agent Panel, click the 3 dots (...) $\rightarrow$ MCP Servers $\rightarrow$ Manage MCP Servers $\rightarrow$ View raw config (opens mcp_config.json).

07

Paste Config & Save Password

Paste the copied JSON configuration into mcp_config.json. Replace replace-with-your-application-password with your generated Application Password, then save the file.

08

Verify Connection and Run Your First Prompt

In Antigravity IDE, click Refresh under Manage MCP Servers. You should see wsp-wordpress-mcp listed as enabled! Open Antigravity Agent, choose a Gemini Pro model, and prompt: "Please create an Elementor page for a website design agency."

Frequently Asked Questions

Is connecting Google Antigravity 2.0 to WordPress free?

Yes! Both the WSP WordPress MCP plugin and the MCP Adapter are 100% free and open-source under the MIT license. Antigravity IDE also offers free model tiers.

Can Antigravity create Elementor layouts directly on my site?

Yes. Once connected, Antigravity Agent can create Elementor pages, build section containers, and insert headings, buttons, and placeholder media directly inside your WordPress site.

Do I need coding skills to connect Antigravity with WordPress?

No coding required. The entire process involves installing two plugins, copying a JSON configuration file, and adding a WordPress Application Password.

Which AI models in Antigravity work best with WordPress MCP?

We recommend selecting Gemini 3.1 Pro (High) or Gemini 3.5 Pro in Antigravity for better multi-step reasoning and precise Elementor layout generation.
